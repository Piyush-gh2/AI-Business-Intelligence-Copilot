# AI Business Intelligence Copilot

AI Business Intelligence Copilot is a simple AI-powered dashboard built using Python, Pandas, and Streamlit.  
The application allows users to upload business datasets and automatically generate insights from the data.

## Features

- Upload business datasets in CSV format
- Automatic data preview
- Business insights generation
- Sales analysis (Total, Average, Maximum)
- Simple and interactive dashboard using Streamlit

## Technologies Used

- Python
- Streamlit
- Pandas
